# insurance-analytics-dashboard
End-to-end Insurance Analytics Dashboard using SQL and Power BI for KPI tracking, trend analysis, and customer segmentation.
🚀 **Built an End-to-End Insurance Analytics Dashboard using SQL & Power BI**

I recently worked on a project where I transformed raw insurance data into meaningful business insights using **SQL and Power BI**.

🔹 **What I did:**

* Cleaned and structured raw data using SQL
* Built analytical views for KPIs, trends, and segmentation
* Connected the data to Power BI for visualization
* Designed an interactive dashboard for decision-making

📊 **Dashboard Highlights:**

* Total Policies Issued: **14K+**
* Average Policy Duration: **~813 days**
* Customer Segmentation by Gender
* Monthly & Yearly Policy Trends
* Policy Duration Analysis

💡 **Key Insights:**

* Majority of policies are concentrated within a single customer segment
* A noticeable spike in policy issuance was observed around a specific year
* Most policies fall under short-term duration categories

🛠️ **Tools Used:**
SQL | Power BI | Data Modeling | Data Visualization
